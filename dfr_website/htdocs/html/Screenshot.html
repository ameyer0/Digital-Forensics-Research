Using a raw image as input for the Deleted File Recovery tool (Autopsy)

Step 1. Enter your case name and select the directory you want to store the results.

Step 2. Enter optional information for example, the case number, name, email etc.

Step 3. Select the type of data source which you will input to the tool.

Step 4. Select the data source you want to input and analyze (select the raw image you want to input). For example, in our case, case_2.raw.

Step 5. Select Ingest Modules that you may need to help analze the raw image.

Step 6. Select deleted files and look if the file that was deleted was able to be recovered.

Step 7. Click File Meta data to view more details about the file that was deleted.
