Incorporating NIST Standards in Digital Forensics Curriculum

A collection of hands-on activities, course modules, and research findings.

There is a set of NIST standards for deleted file recovery tools that can be used to evaluate the performance of these tools. Some hands-on lab activities were created in order to evaluate these deleted file recovery tools. The methods used in these labs to evaluate each tool is presented in this section, along with the results from each test.
