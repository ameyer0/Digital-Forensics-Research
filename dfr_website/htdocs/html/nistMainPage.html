Incorporating Standards Education in Digital Forensics Curriculum

A collection of hands-on activities, course modules, and research findings.

NIST has compiled a set of standards for deleted file recovery tools that can be used to evaluate the correctness and performance of these tools. A set of hands-on lab activities to evaluate the DFR tools are available here. The labs can be readily incorporated in a Digital Forensics curriculum.

A hands-on activity to evaluate hardware write blocker (HWB) is created which can be readily incorporated into a Digital Forensics curricula. The NIST CFTT system is used to evaluate a HWB. A sample set of test results is available here, which can be reproduced or compared with.
